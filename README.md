# PromptCraft

**AI-ассистент для строительства в Minecraft**

PromptCraft – это мод для Minecraft (Fabric), который позволяет игрокам создавать структуры в мире с помощью текстовых запросов к искусственному интеллекту. Мод поддерживает интеграцию с AI API для генерации строительных планов на основе естественного языка.

## Основные возможности

- **Текстовое строительство**: Описывайте желаемую структуру текстом, и AI построит её в выделенной области
- **Инструмент выделения**: Специальная кисть (Selection Brush) для выбора области построения
- **Визуальный предпросмотр**: Динамическая визуализация выделенной области в реальном времени
- **Система отмены/повтора**: Возможность отменять и возвращать изменения (undo/redo)
- **Настройки**: Графический интерфейс для настройки API ключа, модели и других параметров

## Требования

- **Minecraft**: 1.20.1
- **Fabric Loader**: >= 0.15.0
- **Fabric API**: Любая версия для 1.20.1
- **Java**: >= 17

## Установка и запуск

Для запуска мода выполните в терминале:

```bash
.\gradlew.bat build
.\gradlew.bat runclient
```

После первого запуска мода в директории конфигурации Minecraft (`config/promptcraft/`) будут созданы файлы.

## Настройка

### config.json
Основной файл конфигурации:
```json
{
  "provider": "nvidia",
  "baseUrl": "https://integrate.api.nvidia.com/v1",
  "model": "meta/llama-3.1-70b-instruct",
  "accessMode": "admins_only",
  "themeColor": "#17b95f",
  "showProcessMessages": true,
  "enableDestructionAnimation": true,
  "showSelectionPreview": true,
  "selectionLimitEnabled": true,
  "maxSelectionWidth": 64,
  "maxSelectionHeight": 64,
  "maxSelectionDepth": 64,
  "maxRepairAttempts": 3
}
```

### .env
Файл для хранения API ключа:
```
NVIDIA_API_KEY=ваш-api-ключ-здесь
```

## Получение API ключа

1. Перейдите на [NVIDIA API Catalog](https://www.nvidia.com/en-us/ai-data-science/nvidia-api-catalog/)
2. Зарегистрируйтесь и получите API ключ
3. Вставьте ключ в файл `.env` или используйте команду `/promptsettings` в игре

## Использование

### Выделение области

1. Возьмите Selection Brush в руки (находится в табл. "PromptCraft")
2. ЛКМ по блоку – установить первую позицию
3. ПКМ по блоку – установить вторую позицию
4. Выделенная область будет отображаться полупрозрачным контуром

### Создание структуры

1. Выделите область с помощью Selection Brush
2. Выполните команду:
   ```
   /promptcraft построй замок с башней и водопадом
   ```
3. Подтвердите запрос, нажав на кнопку `[YES, START]` или выполните `/promptconfirm`
4. Мод очистит выделенную область и построит структуру по вашему запросу

### Команды

| Команда | Описание |
|---------|----------|
| `/promptcraft <текст>` | Создать новую структуру по запросу |
| `/promptedit <текст>` | Отредактировать последний запрос |
| `/promptconfirm` | Подтвердить ожидающий запрос |
| `/promptcancel` | Отменить ожидающий запрос |
| `/promptundo` | Полностью отменить постройку |
| `/promptback` | Откатиться на шаг назад (для правок) |
| `/promptnext` | Вернуть отменённые изменения |
| `/promptsettings` | Открыть настройки мода |

- По умолчанию доступ только для администраторов (уровень прав >= 2)
